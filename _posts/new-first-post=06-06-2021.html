New First Post

06-06-2021


A new first post is something I've written like a hundered times by now. Moving from handwritten html based websites to static site generators like jekyll I've tried a lot of things. But I got bored of writing out all the html by hand, and I needed a break from my latest project. For these reason, I've written a simple tool named lih to help out with quickly setting up a blog.

This site is powered by lih and another tool I wrote some time ago, sitefl Sitefl is a markdown alternative which means just like when I was using Jekyll, I can write posts comfortably.

Lih is not really meant to do a whole lot and there's probably a few bugs in it, so I'll hopefully iron them when I get to them 😁.
